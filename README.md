# 📊 Student Performance Analysis Dashboard

## Overview

This project analyzes student academic performance using Power BI.

The dashboard provides insights into:

- Student performance across Mathematics, Reading, and Writing
- Gender-wise comparison
- Ethnicity-wise analysis
- Impact of parental education
- Test preparation analysis
- Top-performing students
- Interactive filtering using slicers

---

## Dashboard Features

- KPI Cards
- Bar Charts
- Pie Charts
- Interactive Slicers
- Student Performance Comparison
- Insights & Summary Page

---

## Dataset

The dataset contains information for 100 students including:

- Student ID
- Gender
- Ethnicity
- Mathematics Score
- Reading Score
- Writing Score
- Test Preparation
- Parental Education

- Microsoft Power BI
